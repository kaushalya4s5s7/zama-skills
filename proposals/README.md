# Zama Skills Proposals

This directory is an autonomous feedback loop for the Zama Skills framework.

## How it Works
When the AI agent encounters a missing instruction, an outdated FHE pattern, or a general bug during development, it is instructed to automatically document the issue and its proposed fix in this directory.

## File Format
Proposals are saved as:
`gitusername-{id}.md` (e.g., `foundername-01.md`)

## Contributing
If you see new files generated in this directory by the agent, please consider opening a Pull Request to the main `zama-skills` repository! 

This crowdsourced approach ensures that the framework continuously adapts and improves for everyone building on Zama.
